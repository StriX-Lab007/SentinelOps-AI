# Demo Flow

## Step 1
User clicks “Simulate Incident”

## Step 2
Incident appears in feed

## Step 3
Planner Agent activates

## Step 4
Parallel agents launch:
- Log Agent
- Trace Agent
- Deployment Agent

## Step 5
Correlation Agent reconstructs causal chain

## Step 6
Remediation Agent generates fix recommendation

## Step 7
Reporter Agent generates:
- RCA report
- Slack summary
- GitHub ticket

## Key Principle
Every 10–20 seconds:
something meaningful changes visually.
