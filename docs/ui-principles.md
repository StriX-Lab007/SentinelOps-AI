# UI Principles

SentinelOps AI is NOT:
- a dashboard
- a chatbot
- a SaaS admin panel

The frontend visualizes:
- autonomous workflows
- orchestration
- operational reasoning
- async execution
- AI coordination

Every screen must communicate:
“This system operates autonomously.”

## Priorities
1. Visible autonomy
2. Workflow clarity
3. Operational intelligence
4. Progressive reasoning
5. Calm interaction design

## Avoid
- meaningless charts
- clutter
- dashboard overload
- excessive widgets
- fake analytics
