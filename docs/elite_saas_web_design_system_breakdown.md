# 3. VISUAL DESIGN DNA

# A. TYPOGRAPHY SYSTEM

## PRIMARY CHARACTERISTICS

### Large Hero Headlines

Characteristics:

- ultra bold
- high contrast
- extremely short
- emotionally compressed
- visually dominant

Examples:

- “Meet the night shift.”
- “Your shortcut to everything.”

Notice:

They are:

- NOT descriptive
- NOT technical
- NOT feature-heavy

Instead:

They communicate IDENTITY.

This is positioning psychology.

---

## Typography Hierarchy

### Layer 1 — Brand Statement

Very large.

Purpose:

- emotional hook
- positioning
- attention control

Font behavior:

- 700–900 weight
- tight line height
- centered
- low word count

---

### Layer 2 — Contextual Explanation

Smaller supporting text.

Purpose:

- reduce ambiguity
- explain product category
- bridge emotional → rational

Characteristics:

- medium contrast
- 16–22px equivalent
- wide line spacing
- readable but visually secondary

---

### Layer 3 — UI Labels

Tiny.

Purpose:

- functional navigation
- system information

Characteristics:

- muted contrast
- high spacing
- low visual dominance

---

# Typography Strategy Summary

The typography system is built on:

- aggressive hierarchy separation
- attention funneling
- compression of information
- whitespace amplification

# C. SPACING SYSTEM

This is where most amateur design collapses.

# THE WEBSITES USE MACRO SPACING

Meaning:

Large empty areas intentionally.

Why?

Because spacing creates:

- authority
- readability
- focus
- perceived sophistication
- emotional pacing

---

# SPACING ARCHITECTURE

## Section Padding

Massive vertical spacing.

Typical structure:

- 120px–220px between major sections
- 24–40px inside cards
- 8px spacing grid internally

---

## Why It Works

Spacing:

- slows visual chaos
- allows product screenshots to dominate
- improves scanning
- creates cinematic pacing

This is why premium sites feel “expensive.”

They waste space intentionally.

Cheap websites fear whitespace.

## RADIUS

Only 3–4 levels.
