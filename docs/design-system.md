# SentinelOps AI Design System

## Design Philosophy
Raycast-inspired AI-native operating system.

The UI should feel:
- calm
- fluid
- operationally intelligent
- spatially clear
- premium
- continuously alive

Avoid:
- generic SaaS dashboards
- cyberpunk aesthetics
- noisy analytics layouts
- chatbot interfaces

## Visual Direction
Inspired by:
- Raycast
- Linear
- Vercel Dashboard
- LangSmith traces

## Theme
Dark operational interface.
